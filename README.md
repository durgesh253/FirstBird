# SaaS Analytics Dashboard

A modern, premium analytics dashboard for Shopify stores, built with Vanilla JS, Vite, and Chart.js.

## Features

- **Dashboard Overview**: KPI cards, revenue charts, and top performers.
- **Coupons Management**: Track active coupons and their revenue performance.
- **Leads Attribution**: Upload CSV leads and track their conversion funnel.
- **Campaigns Tracking**: Monitor marketing campaigns across different platforms (Instagram, WhatsApp, etc.).
- **Detailed Analytics**: Deep dive into campaign performance with funnel visualizations.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repository (if applicable) or navigate to the project directory.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure

- `src/main.js`: Main application entry and router logic.
- `src/style.css`: Global design system and CSS variables.
- `src/components/`: Reusable UI components (Sidebar, TopHeader, Cards, Charts).
- `src/pages/`: Page-specific logic (Dashboard, Coupons, Leads, Campaigns).

## Technologies

- **Vite**: Fast build tool and dev server.
- **Chart.js**: Interactive charting library.
- **Phosphor Icons**: Premium icon family.
- **Vanilla JS**: Lightweight, no-framework approach.
